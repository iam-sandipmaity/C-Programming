# C Programming Repository

Welcome to my **C Programming** repository! 🚀 This repository contains all my learning and problem-solving experiences while studying the **C programming language**. It serves as a reference for future use and for anyone looking to explore C programming.

## 📌 Repository Overview
- Covers all essential C programming concepts.
- Includes solutions to problems for each topic.
- Well-structured code with comments for better understanding.
- Continuously updated with new learnings and challenges.


## 💡 How to Use This Repository
1. Clone the repository:
   ```sh
   git clone https://github.com/iam-sandipmaity/C-Programming.git
   ```
2. Navigate to the folder:
   ```sh
   cd C-Programming
   ```
3. Explore different topics and try running the programs:
   ```sh
   gcc filename.c -o output && ./output
   ```
